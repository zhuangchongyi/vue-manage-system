## 简单的一个模板模板

[来源](https://github.com/lin-xin/vue-manage-system.git)

## 安装步骤

```
// 把模板下载到本地
git clone https://github.com/zhuangchongyi/vue-manage-system.git
// 进入模板目录
cd vue-admin-template
// 安装项目依赖，等待安装完成之后，安装失败可用 cnpm 或 yarn
npm install

// 开启服务器，浏览器访问
npm run dev

// 执行构建命令，生成的dist文件夹放在服务器下即可访问
npm run build
```
